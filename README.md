# vue-go-back-top

> vue移动端返回顶部组件
# 说明

组件只有一个功能点击返回顶部

# 预览

![demo](https://i.imgur.com/uNdhfpi.gif)

# 使用

    import {GoTop} from "vue-go-back-top";
    //或
    import GoTop from "vue-go-back-top";

    Vue.component(GoTop.name, GoTop)  
    //或
    Vue.use(GoTop)  






## 项目运行

### 安装
	npm install

### 本地运行
	npm run dev

### 构建发布
	npm run build
